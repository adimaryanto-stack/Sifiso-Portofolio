About the Project

The challenge for the Aura Mobile App was to create an intuitive and immersive experience for users seeking mental clarity and stress relief. We designed a fluid user interface with calming visuals, personalized content pathways, and interactive meditation exercises. The solution involved a clean, minimalist design language accented by warm, organic gradients, focusing on user flow to ensure seamless navigation and habit-forming features.
